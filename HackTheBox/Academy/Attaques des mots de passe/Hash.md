# Hash

## Type de hash

Il existe différents hashages qui peuvent identifier grâce à un ID

| **ID** | **Méthode de hash** |
|---|----|
| `$1$` | 	MD5 | 
| `$2a$` | 	Blowfish | 
| `$5$` | 	SHA-256 | 
| `$6$` | 	SHA-512 | 
| `$sha1$` | 	SHA1crypt | 
| `$y$` | 	Yescrypt | 
| `$gy$` | 	Gost-yescrypt | 
| `$7$` | 	Scrypt |
